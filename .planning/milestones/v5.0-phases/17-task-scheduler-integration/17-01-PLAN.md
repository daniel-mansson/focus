<task type="auto">
  <name>Task 1: Add Task Scheduler integration to Inno Setup installer</name>
  <files>installer/focus.iss, installer/test-scheduler.ps1</files>
  <action>
Modify installer/focus.iss to add Task Scheduler integration via Pascal Script:

**[UninstallRun] section** (add before [Code] or at end of declarative sections):
- Add: `Filename: "schtasks.exe"; Parameters: "/Delete /TN ""FocusDaemon"" /F"; Flags: runhidden`
- This handles the simple case. The [Code] section adds a robust fallback with ShellExec "runas".

**[Code] section** -- add all Pascal Script functions:

1. **Global variable**: `var StartupPage: TInputOptionWizardPage;`

2. **BuildTaskXml(AppPath: String; RunElevated: Boolean): String** -- builds the Task Scheduler XML template string.
   - Use the exact XML structure from 17-RESEARCH.md Pattern 1 (LogonTrigger, ExecutionTimeLimit=PT0S, InteractiveToken, MultipleInstancesPolicy=IgnoreNew, etc.)
   - RunLevel: 'HighestAvailable' when RunElevated=True, 'LeastPrivilege' when False
   - Command: AppPath (the {app}\focus.exe path)
   - Arguments: 'daemon'
   - XML declaration: `<?xml version="1.0" encoding="UTF-16"?>`

3. **DetectExistingTask(var IsElevated: Boolean): Boolean** -- checks if FocusDaemon task exists and detects its RunLevel.
   - Use `Exec('schtasks.exe', '/Query /TN "FocusDaemon"', ...)` -- exit code 0 means exists
   - For RunLevel detection: use `Exec(ExpandConstant('{cmd}'), '/C schtasks.exe /Query /TN "FocusDaemon" /XML > "' + TmpFile + '"', ...)` to capture XML output to temp file
   - Use LoadStringFromFile + Pos('HighestAvailable', ...) to detect elevated RunLevel
   - Return True if task exists, set IsElevated to True if HighestAvailable found

4. **InitializeWizard()** -- creates the custom wizard page.
   - `StartupPage := CreateInputOptionPage(wpSelectDir, 'Startup Options', 'Configure how Focus starts.', 'Select startup behavior, then click Next.', False, False);`
   - `StartupPage.Add('Start at logon');`
   - `StartupPage.Add('Run elevated (admin) - Required to navigate between admin windows');`
   - Default: Values[0] := True (start at logon), Values[1] := False (not elevated)
   - Call DetectExistingTask to pre-populate on upgrade: if task exists, set Values[0] := True; if elevated, set Values[1] := True

5. **CurStepChanged(CurStep: TSetupStep)** -- creates/deletes the scheduled task after install.
   - Only act when CurStep = ssPostInstall
   - If StartupPage.Values[0] (start at logon) is True:
     - Build XML via BuildTaskXml with {app}\focus.exe path and StartupPage.Values[1] (elevated) flag
     - Write XML to {tmp}\FocusDaemon.xml via SaveStringToFile
     - Delete existing task first (ignore failure): `Exec('schtasks.exe', '/Delete /TN "FocusDaemon" /F', '', SW_HIDE, ewWaitUntilTerminated, ResultCode)`
     - Create task: build params string `/Create /XML "{tmp}\FocusDaemon.xml" /TN "FocusDaemon" /F`
     - If IsAdminInstallMode (installer already elevated): use Exec directly
     - Else: use ShellExec('runas', 'schtasks.exe', ...) for UAC elevation
   - If StartupPage.Values[0] is False:
     - Try Exec delete first; if fails (ResultCode <> 0), try ShellExec 'runas' delete

6. **CurUninstallStepChanged(CurUninstallStep: TUninstallStep)** -- robust uninstall cleanup.
   - Only act when CurUninstallStep = usPostUninstall
   - Try `Exec('schtasks.exe', '/Delete /TN "FocusDaemon" /F', ...)` first
   - If ResultCode <> 0 (likely needs admin), fall back to `ShellExec('runas', 'schtasks.exe', '/Delete /TN "FocusDaemon" /F', ...)`
   - This supplements the [UninstallRun] entry as a robust fallback

**test-scheduler.ps1** -- create a PowerShell verification script:
- Query task existence: `schtasks /Query /TN "FocusDaemon" 2>$null; if ($LASTEXITCODE -eq 0) { "PASS: Task exists" } else { "FAIL: Task not found" }`
- Query XML for ExecutionTimeLimit: `$xml = schtasks /Query /TN "FocusDaemon" /XML; if ($xml -match 'PT0S') { "PASS: No time limit" } else { "FAIL: Time limit set" }`
- Query XML for RunLevel: `if ($xml -match 'HighestAvailable') { "INFO: Elevated" } elseif ($xml -match 'LeastPrivilege') { "INFO: Standard" }`
- Query XML for LogonTrigger: `if ($xml -match 'LogonTrigger') { "PASS: Logon trigger" } else { "FAIL: No logon trigger" }`
- Query XML for Command path: verify it points to focus.exe with "daemon" argument
- Add a cleanup section: `schtasks /Delete /TN "FocusDaemon" /F` with confirmation prompt
