- Source: `..\focus\bin\Release\net8.0\win-x64\publish\focus.exe`; DestDir: `{app}`; Flags: `ignoreversion`

<task type="auto">
  <name>Task 2: Create build.ps1 orchestrator</name>
  <files>build.ps1</files>
  <action>
Create `build.ps1` at repo root. This PowerShell script orchestrates the full build pipeline.

**Script structure (in order):**

1. **Set strict mode:**
   ```
   $ErrorActionPreference = "Stop"
   ```

2. **Read version from focus.csproj:**
   ```
   [xml]$csproj = Get-Content "focus/focus.csproj"
   ```
   Extract the Version element value (currently "4.0.0"). Handle the case where PropertyGroup may be an array (use `Select-Xml` or direct navigation). Print: "Building Focus v$version"

3. **Run dotnet publish:**
   ```
   dotnet publish focus/focus.csproj -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true
   ```
   Check `$LASTEXITCODE -ne 0` and throw on failure.
   (IncludeNativeLibrariesForSelfExtract=true is CRITICAL -- without it, native DLLs are extracted alongside exe)

4. **Verify single-file output:**
   - Check `focus/bin/Release/net8.0/win-x64/publish/focus.exe` exists
   - Count files in publish directory; warn if more than 1 (indicates native extraction issue)

5. **Compile installer with ISCC:**
   ```
   ISCC.exe /DMyAppVersion="$version" installer/focus.iss
   ```
   Check `$LASTEXITCODE -ne 0` and throw on failure.
   ISCC.exe must be on PATH (per user decision -- no hardcoded path, no parameter override).

6. **Report result:**
   - Check `installer/output/Focus-Setup.exe` exists
   - Print file size in MB
   - Throw if file was not created

**Important constraints:**
- Script assumes it runs from repo root (all paths relative to repo root)
- Do NOT hardcode the ISCC.exe path -- it must be on PATH
- Do NOT add a version bump step -- version is read-only from csproj
- Use backtick line continuation for the dotnet publish command for readability
  </action>

  </verify>
  <done>build.ps1 at repo root reads version from focus.csproj, runs dotnet publish with self-contained single-file flags, invokes ISCC.exe (from PATH) with /DMyAppVersion, and verifies Focus-Setup.exe was created. Script fails fast on any error with descriptive messages.</done>
