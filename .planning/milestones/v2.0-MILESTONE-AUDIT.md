## Milestone Definition of Done

> A persistent background daemon that renders directional overlay previews (colored borders) on candidate windows while CAPSLOCK is held, using a low-level keyboard hook and layered Win32 windows.

**Verdict:** Achieved. The daemon starts with `focus daemon`, hooks CAPSLOCK via WH_KEYBOARD_LL, shows colored border overlays on directional candidates using Win32 GDI layered windows, and dismisses instantly on release.
