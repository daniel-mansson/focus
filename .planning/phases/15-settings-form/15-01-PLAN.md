```csharp
// Load: decompose "#BF4488CC" into alpha byte and Color
static (byte alpha, Color rgb) ParseHexColor(string hex)
{
    var raw = uint.Parse(hex.TrimStart('#'), System.Globalization.NumberStyles.HexNumber);
    byte a = (byte)(raw >> 24);
    byte r = (byte)(raw >> 16);
    byte g = (byte)(raw >> 8);
    byte b = (byte)(raw);
    return (a, Color.FromArgb(r, g, b));  // alpha stored separately
}

// Save: recompose from alpha byte + swatch Color
static string ToHexColor(byte alpha, Color rgb)
    => $"#{alpha:X2}{rgb.R:X2}{rgb.G:X2}{rgb.B:X2}";
```

On load, call `ParseHexColor` on each of the four OverlayColors properties. Store alpha from the Left color (all four share same alpha per design — if they differ, Left wins, documented lossy behavior). Store each direction's RGB in a `Dictionary<Direction, Color> _swatchColors`.

On save, recompose each direction's color using `ToHexColor(_opacityAlpha, _swatchColors[direction])`.

private void OnSettingsClicked(object? sender, EventArgs e)

}
